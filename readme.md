# 干部网络学院高效学习助手

帮助你完成切换章节、课程等繁琐的操作，帮助你更高效、专注地学习课程。

版本：2.0

支持网站：[应急管理干部网络学院](https://yjgb.sset.org.cn)、[广州市干部培训网络学院](https://gzgbjy.gzswdx.gov.cn/)

作者：zzj

## 脚本使用方法

### 安装脚本

1. 打开浏览器**插件**（或**扩展**）界面
2. 打开**开发人员模式**
3. **加载解压缩的扩展**，打开插件所在的文件夹

> 如果没有找到选项，找类似意思的选项

### 使用方法

打开全部要学习的课程的标签页。有以下两种方式：（为了保证切换到下一个课程时不会被其他标签页打断）

1. 打开所有课程后关闭除了课程标签页的所有其他标签页

2. 将除了课程标签页之外的所有标签页移到同一侧，然后从另一侧开始学习

然后，脚本会自动开始播放视频，达到完成进度后自动学习下一节，完成全部小节后关闭标签页，从而学习下一个课程，从而使用户更加高效、专注地学习。

## 其他注意事项

1. **应急管理干部网络学院（yjgb.sset.org.cn）** 网站的课程中包含考试，脚本会自动跳过并且关闭标签页，因此想要做考试的话必须禁用脚本再打开完成
